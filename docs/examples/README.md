# Примеры HTML-отчётов

На **странице файла `.html` в GitHub** (вкладка с кодом) страница **не рендерится** — показываются теги и текст как исходник. Так устроен просмотр репозитория (безопасность).

**Как посмотреть отчёт как веб-страницу:**

1. **[Предпросмотр в браузере (HTMLPreview)](https://htmlpreview.github.io/?https://raw.githubusercontent.com/kobyzev-yuri/nyse/main/docs/examples/signal_MU_2026-04-08_17-52.html)** — откроется тот же вид, что у бота (блоки ①–⑦).
2. Скачать файл **`signal_MU_2026-04-08_17-52.html`** и открыть двойным щелчком локально.
3. [Прямая ссылка Raw](https://raw.githubusercontent.com/kobyzev-yuri/nyse/main/docs/examples/signal_MU_2026-04-08_17-52.html) — в некоторых браузерах покажет исходник; для просмотра используйте пункт 1 или 2.

Файл соответствует разбору в [`../СвоимиСловами.md`](../СвоимиСловами.md).
